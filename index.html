ANALYSIS AND VISUALIZATION OF JOBS DATA SET FOR DATA ANALYSIS JOBS

This project involves the analysis and visualization of a dataset containing job postings for data analysis roles, aiming to uncover trends, skill demands, geographical distribution, and salary patterns. Key objectives include data cleaning and preparation to ensure dataset accuracy, exploratory data analysis to compute key metrics and identify trends, and in-depth skills analysis to determine the most sought-after qualifications. Additionally, the project features geographical mapping to highlight regional job demand and salary analysis to compare compensation trends across different roles and locations. Tools used include Python libraries such as Pandas and NumPy for data manipulation, Matplotlib, Seaborn, and Plotly for visualization, and Geopandas for geographic analysis. The impact of this project lies in providing job seekers with insights to better align their skills with market demands, and offering recruiters valuable data to refine their hiring strategies and understand competitive trends. By integrating these analyses, the project supports strategic decision-making and enhances understanding of the data analysis job market.

MULTI PAGE WEB SCRAPPER

This project involves developing a robust multipage web scraper designed to systematically extract and analyze data on the Most Valuable Player (MVP) for each year from various annual award listings. The objectives include constructing a scraper to navigate through multiple web pages, extract MVP names, award years, and relevant statistics, and consolidate this data into a structured format for comprehensive analysis. Tools used include Python libraries such as BeautifulSoup and Scrapy for web scraping, and Pandas for data organization. The impact of this project lies in providing a thorough historical record of MVP selections, enabling deeper insights into player achievements over time and supporting fans, analysts, and researchers in understanding trends and patterns in MVP awards.
